Class: createEventModel

createEventModel(_name, _inputsopt) → {Object}

new createEventModel(_name, _inputsopt) → {Object}

A Factory for creating an event object model (json). This object later is used as a call for the event, so the event is created automatically based in the inputs.

Usage
// First option
const event = createEventModel("myEvent");
event.setInput("string", "name")
.setInput("uint", "count");

// Second option
const inputs = [
    {
      name: "name",
      type: "string",
    },
    {
      name: "count",
      type: "uint",
    },
  ]);
const eventModel = createEventModel("myEvent",inputs);
