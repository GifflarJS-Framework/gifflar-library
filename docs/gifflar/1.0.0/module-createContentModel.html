Module

createContentModel

A Factory for creating an content object model (json). This object is used for creating all content statements like ifs, assignments... It is most used only for components which has a content (function, if, ...).

Author:
  • Levy Santiago

The content object model.

Object
Example
Usage
const contentModel = createContentModel();

Return
{
  content: []
  beginIf: [Function],
  beginElseIf: [Function],
  beginElse: [Function],
  endIf: [Function],
  endElse: [Function],
  endElseIf: [Function]
  setCallEvent: [Function]
  setAssignment: [Function],
  setVariable: [Function]
}
